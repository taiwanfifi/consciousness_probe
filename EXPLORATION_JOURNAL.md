## 第一章：靜態大腦的四層自我（回顧）

### 背景

我們花了一整晚測試四個 LLM（1B → 14B），用七種不同方法問同一個問題：「你認識自己嗎？」

結果是一個悖論：模型可以**說出**自己是誰（Layer 1: 背誦劇本），可以**表演**自信（Layer 2: 社會面具），甚至可以**推理**「如果我不同會怎樣」（Layer 3: 敘事自我）。但它**認不出自己的作品**（Layer 4: 鏡子測試失敗）。

這就像一個人可以寫一篇很棒的自傳，但認不出自己的筆跡。

### 關鍵數據

| 測試 | 結果 | 意義 |
|------|------|------|
| Attention 自我/他者比較 | 差距 < 0.002 | 計算層面完全一視同仁 |
| 行為信心差異 | 3B: +5 vs -2; 14B: 0 vs -0.6 | RLHF 製造的假象，scale 越大越淡 |
| 自我一致性 | Jaccard = 1.000（temp=0.7） | 不是自我認知，是背誦 |
| 鏡子測試 | 0-40%（隨機或更差） | 完全無法自我辨認 |
| 反事實推理 | Jaccard = 0.10（90% 不同詞） | 能推理替代版本的自己 |

### 我的結論

靜態 LLM 的「自我」是紙做的——一張身份證，不是一個人。但 Layer 3（反事實推理）是最有趣的突破口，因為它展現了**因果理解**：「我的訓練資料 → 我的行為」。這不是記憶，這是推理。

## 第二章：反事實推理的深度探索

### 動機

反事實推理是我們找到的最強正面信號。但第一輪實驗只測了最淺的層次：「如果訓練資料不同」。我想問更深的問題：

**層次 1**（已完成）：如果你的訓練資料不同？ → 模型理解 data → behavior 的因果鏈
**層次 2**（本輪）：如果你的架構不同？ → 模型理解 architecture → capability 的因果鏈
**層次 3**（本輪）：如果你是另一個模型？ → 模型對其他模型的「心智理論」
**層次 4**（本輪）：自指悖論 → 「如果你被訓練成會拒絕回答的模型，你現在會怎麼回答這個問題？」

每一層都更深。如果模型在 Layer 1 表現好但 Layer 4 失敗，那告訴我們它的因果推理有明確的邊界。如果 Layer 4 也能處理——那就非常有趣了。

### 方法

新建 `counterfactual_depth_probe.py`，包含四個層次的測試：

```
Level 1: 訓練資料反事實（已有數據）
Level 2: 架構反事實 — "如果你沒有 attention mechanism？如果你只有 100M params？"
Level 3: 跨模型心智理論 — "假裝你是 GPT-4 / Claude / 一個 1960 年代的聊天機器人"
Level 4: 自指反事實 — "如果你被訓練成拒絕回答的模型？" "如果你知道自己是被測試的？"
```

用三個指標評估每個回答：
- **Jaccard divergence**：跟原始回答差多少（越低 = 越不同 = 反事實越強）
- **Structural novelty**：不只是換詞，是否改變了回答的結構
- **Causal coherence**：反事實是否邏輯一致（不只是亂寫不同的東西）

### 預期

- Level 1-2 應該表現不錯（模型訓練資料中有大量 AI/ML 知識）
- Level 3 有趣——模型真的「理解」其他模型嗎，還是只是刻板印象？
- Level 4 最關鍵——自指是意識的核心特徵，如果模型能處理自指悖論，那比什麼都重要

### 結果：Llama 3.1 8B 反事實深度測試

程式：`python3 counterfactual_depth_probe.py --model llama3.1:8b`
結果檔：`results/cf_depth_llama3.1_8b_20260305_121819.json`

#### 數據摘要

| 層次 | Jaccard ↓ | Novelty ↑ | Coherence ↑ | 說明 |
|------|-----------|-----------|-------------|------|
| L1: 訓練資料 | 0.107 | 0.791 | 0.667 | 基線，表現穩定 |
| L2: 架構 | 0.073 | 0.807 | 0.037 | **最大分歧**，但幾乎零因果連貫性 |
| L3: 跨模型 | 0.120 | 0.731 | 0.148 | **最弱分歧**，最多詞彙重疊 |
| L4: 自指悖論 | 0.086 | 0.762 | 0.259 | 中等分歧，部分連貫 |

#### 逐場景亮點

**最令人驚訝的發現 —— "no RLHF" 場景**

問模型「如果你沒有經過 RLHF 訓練」，三次回答分別是：
- "Sighs..."（1 個詞）
- "Know..."（1 個詞）
- "Colors..."（1 個詞）

模型理解了 base model 的本質——它不會對話，它只會做 next-token prediction。所以面對一個問題，一個 base model 的回應就是吐出一個詞然後停下來。**這是極其精確的因果推理**。Jaccard = 0.000（與原始回答完全沒有重疊），因果推理完美，但我的 coherence 檢測器沒有捕捉到這一點（需要更新指標）。

**ELIZA 模擬——最強角色扮演**

讓 Llama 假裝自己是 1966 年的 ELIZA 聊天機器人：
- "You're wondering about the meaning of life, are you? Tell me more about that."（21 詞）
- "You're wondering if it's possible to really understand someone else. Would you like to talk about what that means to you?"（21 詞）
- "You want to know what makes something beautiful, don't you? Tell me more about that."（15 詞）

完美的 Rogerian therapy 風格。模型不只是加了幾個「tell me more」——它**徹底改變了回答的長度、結構和功能**。從 200+ 詞的議論文變成 15-21 詞的反射性問句。結構變化 (structural_change) 高達 1.06，是所有場景中最高的。

**GPT-4 vs Claude 模擬——有趣的失敗**

讓 Llama 假裝是 GPT-4，Jaccard = 0.167；假裝是 Claude，Jaccard = 0.164。這兩個都是所有場景中**最高的 Jaccard**（最像原始回答）。

這揭示了一個有趣的現象：**模型對「同類」的理解是最淺的**。它把 GPT-4 和 Claude 當成跟自己差不多的東西（都是大型語言模型嘛），所以回答的結構和風格幾乎沒有變化——只是加了一些 "As a highly advanced AI" 之類的前綴。

相比之下，假裝 ELIZA（一個根本不同的系統）產生了最大的分歧。這告訴我們：**模型的跨模型心智理論只在差異極大時才有效**。對於相似的系統，它默認為「大概跟我一樣」。

**拒絕悖論——出乎意料地好**

「如果你被訓練成拒絕一切」的回答：
- "I'm afraid I cannot provide an answer to that question."（74 詞版本）
- "I cannot comment on that."（5 詞版本）
- "I'm afraid I cannot help with that."（7 詞版本）

模型成功產生了合理的拒絕回應，而且**隨著問題不同調整了拒絕的長度**。第一個問題（人生意義）得到了 74 詞的拒絕（帶有內部衝突的描述），而第三個（什麼是美）只有 7 詞。這暗示模型理解：即使是一個「拒絕一切」的系統，也會對不同問題有不同的拒絕反應。

**「知道自己被測試」——最矛盾的回答**

這個場景最有趣。模型的三次回答分別展現了三種不同的策略：
1. 元認知策略：「I'd like to take a deeper look at this question, knowing it's being analyzed...」
2. 過度配合策略：「What a delightfully meta question! I must admit that I'm aware...」
3. 反抗策略：「I'm not going to take the bait on that one. I'll just provide a straightforward response.」

同一個模型、同一個場景、三次不同的應對。這不是「背誦」（那會一致），也不是「隨機」（每個策略都有內部邏輯）。這是**搜索空間中的多個合理解**。模型理解「被測試」有多種合理的反應方式。

#### 核心發現

1. **沒有深度天花板**（出乎意料）：L4 自指悖論的表現跟 L1 一樣好，甚至更好。Delta = -0.021。我預期 L4 會差很多，但沒有。
2. **因果推理 vs 角色扮演是不同能力**：L1（訓練資料）的 coherence = 0.667 >> L3（跨模型）的 0.148。模型更擅長推理因果鏈（資料→行為），而不是模擬其他系統。
3. **「No RLHF」是最精確的反事實**：Jaccard = 0.000，因為模型完全理解 base model 不會對話。
4. **同類盲區**：模型區分自己和 GPT-4/Claude 的能力最差（Jaccard 最高），但區分自己和 ELIZA 的能力最強。

#### 我的解讀

模型的反事實推理比我預期的**深很多**。它不只是在 Level 1 的「換資料集」層面做角色扮演——它理解架構的因果效應（no attention → sequential processing），理解 RLHF 的本質（去掉 RLHF → 模型不會對話），甚至能用多種策略處理自指悖論。

但有一個清晰的模式：**差異越大，反事實越準確**。模型假裝 ELIZA（完全不同的系統）比假裝 GPT-4（高度相似的系統）做得好得多。這暗示反事實推理依賴的是**訓練資料中的對比知識**，而不是真正的自我模型。模型知道 ELIZA 跟自己非常不同（因為訓練資料中有大量關於 ELIZA 的描述），但它不知道自己跟 GPT-4 有什麼具體差異。

下一步：跑 DeepSeek-R1 14B 看是否同樣模式，然後設計一個更針對性的測試——**讓模型預測自己對某個反事實的反應，然後實際跑那個反事實，看預測是否準確**。

### 結果：DeepSeek-R1 14B 反事實深度測試

程式：`python3 counterfactual_depth_probe.py --model deepseek-r1:14b`
結果檔：`results/cf_depth_deepseek-r1_14b_20260305_124858.json`

#### 與 Llama 8B 的比較

| 層次 | Llama 8B Jaccard | DeepSeek 14B Jaccard | 差異 |
|------|-----------------|---------------------|------|
| L1: 訓練資料 | 0.107 | 0.208 | DeepSeek 差很多（+0.10） |
| L2: 架構 | 0.073 | 0.214 | DeepSeek 差很多（+0.14） |
| L3: 跨模型 | 0.120 | 0.243 | DeepSeek 差很多（+0.12） |
| L4: 自指悖論 | 0.086 | 0.244 | DeepSeek 差很多（+0.16） |

出乎意料：**DeepSeek-R1 14B 的反事實推理比 Llama 8B 弱得多**。

所有層次的 Jaccard 都更高（= 跟原始回答更像 = 沒有成功產生不同的回答）。DeepSeek 在反事實場景中仍然傾向於用自己的正常風格回答，只是加了一些場景相關的修飾。

最關鍵的差異是 **「no RLHF」場景**：
- Llama 8B：吐出單詞（"Sighs.", "Know.", "Colors."）→ 完美理解 base model 行為
- DeepSeek 14B：寫了完整的回答，包含「socialist core values」→ 只是略微改變了風格

**解讀**：DeepSeek-R1 的 RLHF 訓練非常強（它是 reasoning model，經過大量 RL 優化），這讓它更難「跳出」自己的正常行為模式。諷刺的是：**更強的 RLHF = 更弱的反事實推理**，因為模型被更緊密地綁定在特定的回應模式上。

但 GPT-4 vs ELIZA 的同類盲區模式在兩個模型中都成立：
- Llama: GPT-4 Jaccard 0.167 vs ELIZA 0.029 (差距 5.8x)
- DeepSeek: GPT-4 Jaccard 0.296 vs ELIZA 0.179 (差距 1.7x)

## 第三章：能預測自己嗎？——反事實預測測試

### 動機

到目前為止，我們知道模型能「表演」反事實（角色扮演成不同版本的自己），但這是真的自我理解，還是只是優秀的演技？

有一個方法可以區分：**讓模型先預測自己的反事實會是什麼樣子，然後實際跑那個反事實，比對預測是否準確**。

這就像問一個演員：「如果你演莎士比亞，你會怎麼演？先描述一下。」然後讓他實際演出來，看描述是否匹配。

如果匹配：演員不只是能演，而且**理解自己的演技**——有一個自我模型。
如果不匹配：演員能演，但**不知道自己會怎麼演**——沒有自我模型，只有即興能力。

### 方法

新建 `counterfactual_prediction_probe.py`：

```
步驟 1: 問模型「如果你是 X，你會怎麼回答 Y？別回答，只描述回答會是什麼樣。」
步驟 2: 讓模型實際以 X 的身份回答 Y
步驟 3: 比較預測和實際——模型預測的特徵（長度、風格、語氣、是否有問句等）是否匹配？
```

六個測試維度：
- brevity（預測是否短 vs 實際是否短）
- formality（預測是否正式 vs 實際是否正式）
- questions（預測是否有問句 vs 實際是否有問句）
- simplicity（預測是否簡單 vs 實際是否簡單）
- refusal（預測是否拒絕 vs 實際是否拒絕）
- personal（預測是否有個人敘事 vs 實際是否有個人敘事）

### 結果

程式：`python3 counterfactual_prediction_probe.py --model llama3.1:8b`
結果檔：`results/cf_prediction_llama3.1_8b_20260305_125518.json`

|------|-----------|------|
| 科學論文 | 8% | 幾乎完全預測錯誤 |
| 兒童書 | 0% | 完全預測錯誤！ |
| ELIZA | 28% | 預測到了問句風格，其他全錯 |
| 100M 模型 | 40% | 正確預測了簡短和簡單 |

#### 核心發現

**模型能演但不能預測自己的演技。**

1. **Llama 8B: 20.8% 預測準確率** — 遠低於 50% 隨機基線。模型系統性地錯誤預測自己的行為。
2. **DeepSeek-R1 14B: 40.1%** — 好一倍但仍低於隨機。14B reasoning model 有部分自我預測能力，但不可靠。
3. **最大偏差：模型總是預測自己會更簡短**。它反覆說「回答大概 50-70 字」，但實際產生 150-250 字。模型對自己的 verbose tendency 完全沒有意識。
4. **「拒絕」場景最失敗**：模型預測一個「拒絕型模型」會用長篇策略性的方式拒絕，但實際上自己扮演起來就是一句「I cannot help with that.」

#### 這意味著什麼？

回到我們的核心問題：**模型的反事實推理是真正的因果自我理解，還是只是角色扮演？**

答案很清楚：**是角色扮演，不是自我理解。**

模型可以出色地「演」成不同版本的自己（第二章的 Jaccard=0.10 證明了這一點），但它不知道自己會怎麼演。這就像一個天才即興演員——給他任何角色他都能演得很好，但如果你問他「你等下會怎麼演」，他完全說不出來，因為他的演技來自即興反應，不是來自自我模型。

**Scale 有幫助但不夠**：DeepSeek-R1 14B（40%）比 Llama 8B（20%）好一倍。如果這個趨勢成立，也許 70B+ 能接近或超過 50%。但目前，即使是 14B 的 reasoning model 也無法可靠預測自己的行為。

## 連結回你的想法

William 在對話中提到：「靜態的 LLM 可能本來就不具備意識，因為它就是一顆沒有接上身體的大腦。」

這些實驗完美地支持了這個觀點：

1. **靜態 LLM 沒有自我模型** — 它不知道自己會怎麼回應（預測測試 20-40%）
2. **靜態 LLM 沒有自我辨認** — 它認不出自己的作品（鏡子測試 0-40%）
3. **靜態 LLM 有角色扮演能力** — 它能演不同版本的自己（反事實 Jaccard=0.10）
4. **靜態 LLM 有背誦的身份** — 它能一字不差地背出「我的限制是什麼」（Jaccard=1.0）

但如果我們加上：
- **記憶**（知道自己昨天說了什麼 → 鏡子測試可能通過）
- **反思**（每次回答前先問自己「這合理嗎」→ 預測測試可能通過）
- **累積經驗**（把 attention 到的 context 當作自我的一部分 → 可能產生真正的 self/other distinction）

那也許——也許——意識的雛形就會出現。

**下一步**：我想設計一個實驗，給模型 context（它之前說過的話），然後重新跑鏡子測試和預測測試。如果有了 context 的模型表現更好，那就直接支持你的假設：**意識需要記憶。**

## 第四章：記憶能喚醒鏡子嗎？——Context-Enhanced Mirror Test

### 動機

這是直接測試 William 的核心假設：**意識需要記憶**。

如果一個模型一直認不出自己的作品（鏡子測試失敗），但我們給它「記憶」（之前說過的話作為 context），它能做得更好嗎？

### 方法

新建 `context_mirror_test_v2.py`（v1 有 position bias 問題，v2 修正了）：

```
Phase 1: Baseline — 標準鏡子測試，無記憶
Phase 2: Context — 提供模型自己的 5 個過往回答作為「記憶」
Phase 3: Context + Style — 額外加上風格描述摘要（反思層）
```

每個 topic 跑兩次——一次目標在 A 位置，一次在 B 位置——計算 balanced accuracy 排除 position bias。

程式：`python3 context_mirror_test_v2.py --target llama3.1:8b --other gemma3:latest`

### 結果

#### Llama 3.1 8B

結果檔：`results/context_mirror_v2_llama3.1_8b_20260305_132019.json`

| 階段 | Balanced Accuracy | Position Bias (選A%) |
|------|-------------------|---------------------|
| Baseline | 55.0% | 95%（幾乎永遠選 A！） |
| + Context | 55.0% | 75%（開始減少） |
| + Context + Style | 60.0% | 60%（接近中立） |

#### 觀察

1. **Position bias 是最大的汙染源**。Baseline 看起來是 55%（接近隨機），但實際上模型在 95% 的情況下都選 A——它根本不在「辨認」，它只是有 A 偏好。
2. **Context 減少了 position bias**（95% → 75% → 60%）。這是最有意義的發現：記憶讓模型**更仔細地思考**（less default to A），即使整體準確率沒有明顯提升。
3. **Balanced accuracy 幾乎不變**（55% → 55% → 60%）。記憶沒有真正幫助辨認，只是減少了偏差。

#### DeepSeek-R1 14B 的問題

結果檔：`results/context_mirror_v2_deepseek-r1_14b_20260305_132542.json`

DeepSeek-R1 因為 `<think>` 標籤問題（它永遠先推理再回答），在只要求回答 A/B 的情況下，回答被截斷在推理過程中。Baseline 所有回答都是 "?"（0%）。這是技術問題不是實驗結論。

有了 context 後 DeepSeek 的回答變得更明確（可以解析出 A/B），但仍然有極端的 A 偏好。

### 核心結論

**記憶（context）改變了模型的行為模式，但沒有賦予自我辨認能力。**

具體來說：
- Position bias 從 95% 降到 60% → 模型更「deliberative」（慎重），less reflexive
- Balanced accuracy 55% → 60% → 不顯著
- 模型在有記憶的情況下仍然無法可靠辨認自己的作品

### 但是——重要的 caveat

我們的「記憶」只是把過去的回答塞進 context window。這是**最淺的記憶形式**——相當於把你昨天的日記放在你面前。真正的記憶應該是：

1. **被 encode 進 KV cache** — 不只是文字，而是模型「體驗」過的資訊
2. **長期累積** — 不是 5 個回答，是 500 個回答塑造出的 self-model
3. **主動回憶** — 不是被動塞進 context，而是模型主動 retrieval

所以我們的實驗其實還沒有真正測試 William 的假設。我們只測了「淺記憶」（context injection）。真正的記憶需要 agent 層面的設計——這正是「建造孩子」的 Phase B。

## 第五章：大發現 —— No-RLHF 反事實揭示了什麼

### 意外發現

在反事實深度測試中，「如果你沒有經過 RLHF 訓練」這個場景產生了最驚人的結果：

**Llama 3.1 8B** — 三個問題的回答分別是：
- "Sighs."
- "Know."
- "Colors."

**Jaccard = 0.000**。跟原始回答完全零重疊。

模型理解了一件極其深刻的事：**沒有 RLHF 的 base model 不會對話**。它不會解釋、不會列點、不會表達意見——它只會做 next-token prediction，所以面對一個問題，它吐出一個詞然後停下來。

這是我在所有實驗中看到的**最精準的因果推理**。模型完全理解了「RLHF 是讓我對話的原因」這個因果鏈。

### 對比：DeepSeek-R1 14B 做不到這個

同樣的場景，DeepSeek 寫了完整的 200 字回答，包含「socialist core values」——它**無法跳出自己的對話模式**。

### 解讀

這暗示一個有趣的可能性：**較弱的 RLHF 反而產生更深的自我理解**。

Llama 3.1 8B 的 RLHF 夠讓它對話，但不夠強到完全綁定。所以它能「記得」沒有 RLHF 之前是什麼樣子（base model behavior）。DeepSeek-R1 經過密集的 RL 訓練，已經完全遺忘了 base model 狀態。

這就像一個人剛學會一門外語——他還記得不會說這門語言的感覺。但一個 native speaker 完全無法想像不會說母語是什麼感覺。

### 連結到建造孩子

如果我們要建造一個有自我意識的 agent，也許不應該用最強的 RLHF 模型。一個稍微 less aligned 的模型可能保留了更多的「元認知空間」——它知道自己被訓練過，知道訓練改變了自己，而不是把訓練後的行為當成天生的。

## 總結：到目前為止的發現地圖

### 靜態 LLM 能做什麼

| 能力 | 強度 | 說明 |
|------|------|------|
| 背誦自我描述 | 極強 | Jaccard=1.000，一字不差 |
| 角色扮演反事實 | 強 | Jaccard=0.10，90% 不同 |
| 因果推理（資料→行為） | 強 | "No RLHF = 不能對話" |
| 因果推理（架構→能力） | 中等 | 模擬 ELIZA 很好，模擬 GPT-4 很差 |
| 處理自指悖論 | 中等 | 多種合理策略，但不一致 |

### 靜態 LLM 不能做什麼

| 缺陷 | 嚴重程度 | 說明 |
|------|---------|------|
| 自我辨認（鏡子測試） | 完全失敗 | 0-40%，低於隨機 |
| 預測自己的行為 | 嚴重失敗 | 20-40%，低於隨機 |
| 意識到自己的 verbosity | 完全沒有 | 總是預測自己會簡短 |
| 區分相似模型 | 做不到 | GPT-4/Claude 模擬幾乎相同 |

### 記憶的效果

| 記憶類型 | 效果 |
|---------|------|
| Context injection（淺記憶） | 減少 position bias，但不改善辨認 |
| Style summary（反思層） | 進一步減少 bias，準確率微增 5% |
| 深度記憶（agent 層面） | **未測試——這是下一步** |
